== Description ==

**What is ‘Summary Card with Large Image’**

TWe are glad to bring Twitter Summary Card with Large Image addon for all of our user. This is one of the newest card, and super useful for website who has content to share and nice image as well.

In one word I will summarize it as Photo Card + Summery Card = Summary Card with Large Image. You get the visual exposure of Photo Card plus content depth of Summery card, all together. Read details in Twitter Official site, we just automate the system and you dont have to code anything! But you must have free Twitter Cards Meta already installed, this is an addon.



== Installation ==

1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Locate the zip file you just downloaded  
3. Install, then Activate it.

.


= How To Enable Twitter Cards: =

1. Activate the Plugin.
2. Follow the easy steps from the plugin Setting page **sidebar**.

You are good to go!

== Frequently Asked Questions ==

= Does it need any theme customization or anything? =

No, just activate the plugin and follow the steps.

= Image dimension? =

Product Card will use square image. If you provide any other size image, it will be cropped. (From Twitter Cards guideline page).

= Do you have more detail guideline? =

Yes, check [this page](http://wpdeveloper.net/free-plugin/twitter-cards-meta/#tab2) for detail guide.
